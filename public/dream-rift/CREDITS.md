# Dream Rift — Asset Credits

External art and audio used by the game, loaded via `manifest.json`. If an asset
is missing, the game falls back to its built-in procedural art/synth.

## Music

- **BGM ("Double Spoiler"-style loop)** — composed by **Frol**
  (https://frol-game-music.itch.io/), distributed free for game use.
  Files: `audio/bgm.ogg`, `audio/bgm.mp3`. Loop metadata from the pack's
  `bgm.txt` (LOOPSTART/LOOPLENGTH).

## Sprites

- **Character + boss sprite sheets** (`sprites/players/*`, `sprites/bosses/*`)
  and **bullet/particle sheets** (`bullets/*`) were provided by the project
  owner. They are Touhou-style danmaku sprites (64×64 cells, 4-column sheets).

> Note for maintainers: only keep assets here whose license permits
> redistribution in this repository. Verify each pack's terms before committing.
