§ 03 Practice

Studio

Every design is a specification. A screen and a shelf pose the same question.

Constraints, a grid, a material, a tolerance — and the same discipline in answering. We treat light and matter as one continuous medium and render each with equal precision. Color, here, is light: anything that glows is a screen, anything matte is an object. Only one piece is allowed to break the rule — because it makes light of its own.

Blue = Screen (light) Graphite = Object (matter) Rule 01 — Color exists only as light
