About this book. This is a teaching text. It uses the production rmhstudios.com codebase as a running case study to introduce the ideas behind modern full-stack architecture: SSR and hydration, ORMs at scale, realtime state, the strangler-fig migration pattern, hermetic builds, and horizontal scaling. Each chapter opens with learning objectives and closes with exercises; three practice exams and full worked solutions appear in the back matter.

A note on accuracy. Every architectural claim is grounded in the real repository — real file paths, ports, event names, models, and scripts. Where a component is scaffolded rather than production-hardened, the text says so.
