The complete configuration surface of the Go fleet: every environment variable any binary reads, the client/metrics ports each service binds, the WebSocket path prefixes the gateway routes, and a note on the single reused production Secret. All names and defaults are taken from pkg/config/config.go, each cmd/*/main.go, and internal/gateway/proxy.go. The fleet is intentionally a drop-in for the legacy Node services: it reads the SAME .env.production the Helm chart from PR #121 already mounts (DATABASE_URL, the PORT_* family, DISCORD_*, DEEPSEEK_*, BETTER_AUTH_*).

The reused .env.production Secret

The chart does not define its own credentials. The env.secretName value defaults to rmhstudios-secrets — the exact Kubernetes Secret PR #121's deploy flow already creates out-of-band with kubectl create secret --from-env-file .env.production. The Go fleet reads the SAME variables (DATABASE_URL, DISCORD_*, DEEPSEEK_*, BETTER_AUTH_*), so no new secret management is introduced.

# deploy/helm/rmhstudios-go/values.yaml
env:
  secretName: rmhstudios-secrets   # created by PR #121 from .env.production
deploy/deploy-go.sh is parallel to PR #121's deploy-k8s.sh and deliberately reuses the same Secret and the same single-node-vs-registry switch; it expects the rmhstudios (SSR web) chart to be up first.
