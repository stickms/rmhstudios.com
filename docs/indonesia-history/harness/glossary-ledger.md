# Glossary Ledger — *Indonesia: A Comprehensive History*

**Append-only.** Define a term the first time any chapter uses it; later chapters reuse the
definition. This ledger feeds the back-matter Glossary. Keep alphabetical.

| Term | Definition |
|------|-----------|
| adat | Customary law and tradition of local communities; coexisted with Islamic and colonial law. |
| Dwifungsi | "Dual function" — New Order doctrine giving the military both defense and socio-political roles. |
| Konfrontasi | Sukarno-era armed "Confrontation" opposing the formation of Malaysia (1963–66). |
| kraton | A Javanese royal palace/court (e.g., the kratons of Yogyakarta and Surakarta). |
| Pancasila | The five founding principles of the Indonesian state (belief in God; humanity; unity; democracy; social justice). |
| pribumi | "Native"/indigenous Indonesians, as distinct from foreign-descended groups; a fraught colonial-era category. |
| Reformasi | The post-1998 era of democratic reform following Suharto's fall. |
| Sumpah Pemuda | The 1928 "Youth Pledge" affirming one homeland, one nation, one language (Indonesian). |
| thalassocracy | A state whose power rests on control of the sea and maritime trade (e.g., Srivijaya). |
| VOC | *Vereenigde Oostindische Compagnie*, the Dutch East India Company (1602–1799). |
| wayang | Traditional Javanese/Balinese puppet theatre (esp. *wayang kulit*, shadow puppetry). |
